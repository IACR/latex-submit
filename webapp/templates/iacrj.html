The iacrj document class

Up until 2026 we were using a document class called iacrcc.cls for the IACR Communications in Cryptology. We are now in the process of switching to a new document class iacrj.cls that can be used for all three journals CiC, ToSC, and TCHES. This document class requires an additional file metacapture.sty that is included in the download.

Download

You can download the LaTeX files in a zip file. This download includes the LaTeX document class iacrj.cls, the package metacapture.sty, and the documentation.
