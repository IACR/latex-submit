The iacrcc document class

Papers for the IACR Communications in Cryptology must be prepared using the iacrcc LaTeX document class. This file provides a crucial part of the publishing pipeline, because it allows us to extract all the metadata required for open access publishing.

Download

You can download the LaTeX files in a zip file. This download includes the LaTeX document class, a template to start from, and documentation. There are also quick start guides to convert from llncs, convert from ieeetran, and convert from iacrtrans.
