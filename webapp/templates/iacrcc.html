The iacrcc document class

Papers for the IACR Communications in Cryptology must be prepared using the iacrcc LaTeX document class and iacrcc BibTeX style. These files provide a crucial part of the publishing pipeline, because they allow us to extract all the metadata required for open access publishing.

You can download the LaTeX files in a zip file. This download includes the LaTeX document class, the bibliography style, a template to start from, and documentation.
