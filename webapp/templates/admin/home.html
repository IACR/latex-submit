You are logged in as {{current_user.name}} with email {{current_user.email}}.

This provides access to administrative actions by Journal editors and copy editors. The view you see depends on the permissions that you have.

  1. Admins can see everything.
  2. Journal editors can see their journals.
  3. Journal copyeditors can see the journals they are responsible for.

Contact us at {{config.SITE_CONTACT_EMAIL}} if you have any questions.

This server fulfills three steps in a journal publishing workflow:

  1. Upload of final LaTeX for compilation on this server.
  2. Copy editing of the final versions on this server.
  3. Publishing to the publicly-facing site for the journal.

In order for a journal to use this, it must use a document class that loads the metacapture LaTeX package. That includes IACR journals that use the iacrj document class.
